Main Classes

SIMPLE_SCULPTOR

Purpose: Facade for complete text-to-3D pipeline

Creation:
make

Configuration (Builder Pattern):
set_device (device: STRING): like Current
set_voxel_size (size: REAL [0.1..1.0]): like Current
set_seed (seed: INTEGER): like Current
set_num_inference_steps (steps: INTEGER [16..256]): like Current
set_model_path (path: STRING): like Current

Model Lifecycle:
load_model
unload_model
is_model_loaded: BOOLEAN
estimated_inference_time: REAL

Generation:
generate (prompt: STRING): SCULPTOR_RESULT
batch_generate (prompts: LIST [STRING]): LIST [SCULPTOR_RESULT]
generate_and_view (prompt: STRING): SCULPTOR_RESULT

Invariants:
engine_not_void
config_not_void

SCULPTOR_ENGINE

Purpose: ONNX Point-E inference engine

Creation:
make

Configuration:
set_device (device: STRING ["CPU", "CUDA", "TensorRT"])
set_model_path (path: STRING)

Model Management:
load_model (requires: model_path_set, not_already_loaded)
unload_model (requires: is_loaded)
is_model_loaded: BOOLEAN

Inference:
execute (prompt: STRING; seed: INTEGER): SCULPTOR_INFERENCE_RESULT
estimated_inference_time: REAL

Status:
device: STRING
model_path: detachable STRING

Invariants:
device_valid (must be CPU, CUDA, or TensorRT)

SCULPTOR_CONFIG

Purpose: Configuration builder with fluent API

Creation:
make (defaults: voxel_size=0.5, seed=42, steps=64)

Configuration:
set_voxel_size (size: REAL [0.1..1.0]): like Current
set_seed (seed: INTEGER): like Current
set_num_inference_steps (steps: INTEGER [16..256]): like Current

Access:
voxel_size: REAL
seed: INTEGER
num_inference_steps: INTEGER

Invariants:
voxel_size_valid (0.1 ≤ voxel_size ≤ 1.0)
steps_valid (16 ≤ num_inference_steps ≤ 256)

Data Types

POINT_CLOUD

Purpose: 3D point cloud from ONNX inference

Creation:
make (points: ARRAY [REAL_32])  -- xyz triplets

Access:
points: ARRAY [REAL_32]
point_count: INTEGER

Queries:
get_point (index: INTEGER): SCULPTOR_POINT_3D
bounding_box: BOUNDING_BOX_3D
is_empty: BOOLEAN
points_model: ARRAY [REAL_32]  -- MML query

Invariants:
points_not_void
count_valid (count % 3 = 0)

SCULPTOR_MESH

Purpose: Solid 3D mesh with vertices and triangular faces

Creation:
make (vertices: ARRAY [REAL_32]; faces: ARRAY [INTEGER])
make_empty

Access:
vertex_count: INTEGER
face_count: INTEGER

Queries:
bounding_box: BOUNDING_BOX_3D
validate: MESH_VALIDATION_REPORT
vertices_model: ARRAY [REAL_32]  -- MML query
faces_model: ARRAY [INTEGER]      -- MML query

Export:
to_glb (path: STRING)
to_obj (path: STRING)
to_stl (path: STRING)

Invariants:
vertex_count_non_negative
face_count_non_negative

MESH_CONVERTER

Purpose: Voxel-based point cloud to mesh conversion

Creation:
make (voxel_size: REAL [0.1..1.0])

Conversion:
convert (cloud: POINT_CLOUD): SCULPTOR_MESH
convert_with_smoothing (cloud: POINT_CLOUD; iterations: INTEGER [0..10]): SCULPTOR_MESH

Invariants:
voxel_size_valid (0.1 ≤ voxel_size ≤ 1.0)

Geometry Classes

SCULPTOR_POINT_3D

Purpose: 3D coordinate point

Creation:
make (x, y, z: REAL_32)
make_from_array (coords: ARRAY [REAL_32])

Access:
x, y, z: REAL_32

Queries:
distance_to (other: SCULPTOR_POINT_3D): REAL_32
to_array: ARRAY [REAL_32]

Invariants:
coordinates_finite
x_valid, y_valid, z_valid (NaN checks)

SCULPTOR_VECTOR_3D

Purpose: 3D direction vector

Creation:
make (x, y, z: REAL_32)
make_zero
make_from_points (start, end: SCULPTOR_POINT_3D)

Queries:
magnitude: REAL_32
dot_product (other: SCULPTOR_VECTOR_3D): REAL_32
is_zero: BOOLEAN

Transformations:
scale (factor: REAL_32)
normalize

Invariants:
components_finite
x_valid, y_valid, z_valid (NaN checks)

BOUNDING_BOX_3D

Purpose: Axis-aligned 3D bounding volume

Creation:
make (min_x, min_y, min_z, max_x, max_y, max_z: REAL_32)
make_empty

Access:
min_x, min_y, min_z: REAL_32
max_x, max_y, max_z: REAL_32

Queries:
width, height, depth: REAL_32
volume: REAL_32
contains_point (p: SCULPTOR_POINT_3D): BOOLEAN
center: SCULPTOR_POINT_3D

Invariants:
bounds_consistent (min ≤ max)
coordinates_finite
All coordinates pass NaN checks

Result Types

SCULPTOR_RESULT

Purpose: Success/error result (XOR pattern)

Creation:
make_success (mesh: SCULPTOR_MESH)
make_failure (message: STRING)

Access:
is_success: BOOLEAN
mesh: detachable SCULPTOR_MESH  -- Valid if is_success
error_message: STRING

Queries:
summary: STRING

Invariants:
success_xor_error (exactly one of success or error)
success_has_mesh (if success, mesh /= Void)
failure_implies_error (if failure, error non-empty)

SCULPTOR_INFERENCE_RESULT

Purpose: ONNX inference result (success/error XOR)

Creation:
make_success (points: POINT_CLOUD)
make_failure (message: STRING)

Access:
is_success: BOOLEAN
points: detachable POINT_CLOUD  -- Valid if is_success
error_message: STRING

Invariants:
success_xor_error (exactly one of success or error)
success_has_points (if success, points /= Void)

MESH_VALIDATION_REPORT

Purpose: Mesh validation results

Creation:
make

Access:
is_valid: BOOLEAN
error_messages: LIST [STRING]

Queries:
error_count: INTEGER
has_errors: BOOLEAN

Modification:
add_error (message: STRING)
set_mesh_stats (vertex_count, face_count: INTEGER)

Invariants:
valid_xor_errors (is_valid XOR has_errors)

See Quick API for usage examples.
