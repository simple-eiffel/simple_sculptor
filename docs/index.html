Overview

simple_sculptor is a production-ready text-to-3D generation library for Eiffel. It leverages Point-E ONNX inference to convert text prompts into 3D point clouds, then procedurally converts them to solid meshes via voxel-based geometry.

Part of the Simple Eiffel ecosystem.

Features

  • Text-to-3D generation via Point-E ONNX model
  • Multi-device support (CPU, CUDA, TensorRT)
  • Efficient point cloud processing
  • Voxel-based mesh conversion
  • Configurable inference parameters
  • Design by Contract throughout
  • SCOOP-compatible concurrency support
  • Multi-format export (GLB, OBJ, STL)

Quick Example

Generate a 3D mesh from a text prompt:

local
    l_sculptor: SIMPLE_SCULPTOR
    l_result: SCULPTOR_RESULT
do
    create l_sculptor.make
    l_sculptor.set_device ("CUDA")
        .set_voxel_size (0.5)
        .set_num_inference_steps (64)
    l_sculptor.set_model_path ("point-e.onnx")
    l_sculptor.load_model
    l_result := l_sculptor.generate ("a red cube")
    if l_result.is_success then
        -- Process mesh
    end
end

For complete documentation, see the User Guide.
